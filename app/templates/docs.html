Evaluation Metrics

Functionality: This web service compares two xAIF files and reports various metrics for assessing their similarity. It is primarily intended for assessing the results of argument mining systems relative to gold standard data, or for calculating inter-annotator agreement between manually annotated files.

Expected Input: Two JSON xAIF files (with nodes and edges) containing identical text fields.

Expected Output: CASS, Macro F1, Accuracy, Text Similarity, Kappa, U-Alpha


Sample command: curl -X POST -F "file1=@file1.json" -F "file2=@file2.json" http://amf-evaluation-metrics.amfws.arg.tech

Both files must have identical text fields.
